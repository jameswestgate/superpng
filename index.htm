Superpng jQuery plugin

The superpng plug-in has been written to provide a flexible png alpha api to developers using jQuery and who need to support IE6.

The script uses either the AlphaImageLoader filter or vml based on an approach by Drew Diller, depending on the type of tag and whether background-position is required

For more information on alpha pngs and IE6, please see http://queryj.com.

Usage

$(selector).superpng(options);
    The value of the options parameter can be as follows:
  • path: define the location of the transparent .gif file used to replace the original png image. Required for all modes of replacement.
  • mode: filter / vml / auto (default) - describes the method used to enable png alpha.
    • filter: The alpha loader fix is used, replacing the existing image with the blank image found in path.
    • vml: vml markup is placed inside the container. Requires a div tag or other block element.
    • auto: A mode is chosen based on the tag type and background-position value. (default)
  • cache: Determines if background images are cached. Default true.

Example 1

Set .png transparency on an <img> tag. Provides the location of the tranparent .gif to use.

$('img').superpng({path: '/images/blank.gif'});

Example 2

Sprite usage. Set png image on a <div> tag with background-position.

$('.png').superpng({path: '/images/blank.gif'});

Example 3

Use the vml method and change background position when the image is clicked.

$('.png').superpng({mode: 'vml', path: '/images/blank.gif'});
			
$('.png').click(function(e) {
								
	$(this).css('background-position', '20px 0');
	$('.png').superpng();
	e.preventDefault();
});

Example 4

Set png image with repeat.

$('.png').superpng({mode: vml, path: '/images/blank.gif'});
