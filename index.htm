Superpng jQuery plugin

The superpng plug-in has been written to provide a flexible png alpha api to developers using jQuery and who need to support IE6.

The script uses either the AlphaImageLoader filter or vml based on an approach by Drew Diller, depending on the type of tag and whether background-position is required

For more information on alpha pngs and IE6, please see http://queryj.com.

Example 1

Set .png transparency on an <img> tag. Provides the location of the tranparent .gif to use.

$('img').superpng({path: '/images/blank.gif'});

Example 2

Set png image on a <div> tag with background-position.

$('.png').superpng({path: '/images/blank.gif'});

Example 3

Use the vml method and change background position when the image is clicked.

$('.png').superpng({mode: 'vml', path: '/images/blank.gif'});
			
$('.png').click(function(e) {
								
	$(this).css('background-position', '20px 0');
	$('.png').superpng();
	e.preventDefault();
});
